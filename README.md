e-plantShopping
Project Name
Paradise Nursery – E-Plant Shopping Application

Description
An e-commerce platform for buying indoor and outdoor plants. Built with React, Redux Toolkit, and React Router.

Features
Product listing with categories (Indoor, Outdoor)
Add to cart, update quantity, remove items
Cart total with dynamic calculation
Responsive background image on landing page
Tech Stack
React 18
Redux Toolkit
React Router DOM
CSS3
